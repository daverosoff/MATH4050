    Subsection 3.3 HW 03

    Note: These statements should be proved using only the definitions and properties presented in the lecture notes, as well as the additional definitions below.
    🔗
    Consider a subset \(A\) of \(\R\text{.}\) We say that \(M \in \R\) is a lower bound for \(A\) if \(x \geq M\) for all \(x \in A\text{.}\) We say that \(M\) is a greatest lower bound for \(A\) if, for all lower bounds \(M'\) of \(A\text{,}\) we have \(M \geq M'\text{.}\)
    🔗
    Greatest lower bounds are unique when they exist, and every nonempty subset of \(\R\) that has a lower bound has a greatest lower bound. You can assume all this without proof for the rest of the HW 03.
    🔗
    The greatest lower bound of a set \(A\) (provided it exists) is also called the infimum of \(A\) and written \(\inf{(A)}\text{.}\)
    🔗

    Exercises Exercises

    1.
    Let \(x, y \in \R\text{.}\) Prove that if \(x, y \gt 0\text{,}\) then \(x + y \gt 0\) and \(xy \gt 0\text{.}\)
    🔗
    Hint.
    Use the definition of a real number as an equivalence class of Cauchy sequences of rational numbers.
    🔗
    🔗
    🔗
    2.
    Let \(a_n\) be a Cauchy sequence of rational numbers and let \(x \in \R\text{.}\) Show that if \(a_n \leq x\) for all \(n \in \N\text{,}\) then \([a_n] \leq x\text{.}\)
    🔗
    Hint.
    Use a proof by contradiction. If \([a_n] \gt x\text{,}\) then there exists \(r \in \Q\) such that \([a_n] \gt r \gt x\) (why?).
    🔗
    🔗
    🔗
    3.
    Suppose \(B \subseteq A \subseteq \R\) are nonempty and that \(A\) has an upper bound.
    1. Show that \(B\) has an upper bound.
      🔗
      🔗
    2. Show that \(\sup{(B)} \leq \sup{(A)}\text{.}\)
      🔗
      🔗
    🔗
    🔗
    4.
    Suppose \(A, B \subseteq \R\) each have an upper bound.
    1. Show \(A \cup B\) has an upper bound.
      🔗
      🔗
    2. Find a formula for \(\sup{(A \cup B)}\text{.}\) Prove your result.
      🔗
      🔗
    🔗
    🔗
    5.
    Let \(A \subseteq \R\) be nonempty and suppose \(A\) has an upper bound. Let \(M = \sup{(A)}\) and define the set \(-A\) by
    \begin{equation*} -A = \{ -x : x \in A \}\text{.} \end{equation*}
    Prove that \(-M = \inf{(-A)}\text{.}\)
    🔗
    🔗
    6.
    Suppose that \(A \subseteq \R\) is nonempty. Show that if \(M\) is a lower bound for \(A\text{,}\) then \(M = \inf{(A)}\) if and only if for every real number \(\eps \gt 0\text{,}\) there is an \(a \in A\) such that \(a \lt M + \eps\text{.}\)
    🔗
    🔗
    7.
    Find the supremum and infimum of the set
    \begin{equation*} \left\{ \frac{m}{m+n} : m, n \in \N \right\}\text{.} \end{equation*}
    Prove your results.
    🔗
    🔗
    8.
    1. Let \(\mathbf{I} = \{ x \in \R : x \not\in \Q \}\) be the set of irrational numbers. Show that if \(x \in \Q\) and \(y \in \mathbf{I}\text{,}\) then \(x + y \in \mathbf{I}\text{.}\)
      🔗
      🔗
    2. Use Item 3.4.1.a and the density of \(\Q\) in \(\R\) to prove that \(I\) is also dense in \(\R\text{.}\) That is, if \(x \lt y\) for \(x, y \in \R\text{,}\) prove that there exists \(t \in \mathbf{I}\) such that \(x \lt t \lt y\text{.}\)
      🔗
      🔗
    🔗
    Hint.
    Apply the density result to \(x - \sqrt{2} \lt y - \sqrt{2}\text{.}\) You may assume \(\sqrt{2} \in \mathbf{I}\text{.}\)

    2.
    1. Show that if \(A\) is a countable set and \(B\) is a finite set with \(A \cap B = \varnothing\text{,}\) then \(A \cup B\) is countable.
      🔗
      🔗
    2. Show that if \(A\) and \(B\) are countable sets, then \(A \cup B\) is countable. (Hint: Write \(B' = B \setminus A\text{,}\) and then \(A \cup B = A \cup B'\) with \(A \cap B = \varnothing\text{.}\))
      🔗
      🔗
    🔗
    🔗
    3.
    Prove that the limit of a convergent sequence is unique: if \((a_n)\) converges to both \(a\) and \(b\text{,}\) then \(a = b\text{.}\)
    🔗
    🔗
    4.
    Use the definition of convergence to prove that \(\lim_{n \to \infty} \frac{2n^2}{n^3 + 3} = 0\text{.}\)
    🔗
    🔗
    5.
    For any real number \(a \geq -1\text{,}\) show that \((1 + a)^n \geq 1 + na\) for all \(n \in \N\text{.}\) (Hint: Use induction on \(n\text{.}\))
    🔗
    🔗
    6.
    1. Let \(x, y \in \R\text{.}\) Use the reverse triangle inequality
      \begin{equation*} \abs{x} - \abs{y} \leq \abs{x - y} \end{equation*}
      to prove the stronger result
      \begin{equation*} \abs{\abs{x} - \abs{y}} \leq \abs{x - y}\text{.} \end{equation*}
      🔗
      🔗
    2. Use the previous part to show that if \((a_n)\) converges to \(a\text{,}\) then \((\abs{a_n})\) converges to \(\abs{a}\text{.}\)
      🔗
      🔗
    🔗
    🔗
    7.
    Suppose \((a_n)\) is a sequence with \(a_n \geq 0\) for all \(n \in \N\text{.}\)
    1. If \((a_n)\) converges to \(0\text{,}\) prove that \((\sqrt{a_n})\) converges to \(0\text{.}\)
      🔗
      🔗
    2. If \((a_n)\) converges to \(a \ne 0\text{,}\) prove that \((\sqrt{a_n})\) converges to \(\sqrt{a}\text{.}\) (Hint: The relation
      \begin{equation*} \abs{\sqrt{x} - \sqrt{y}} = \frac{\abs{x - y}}{\sqrt{x} + \sqrt{y}} \end{equation*}
      may be useful.)
      🔗
      🔗
    🔗
    🔗
    8.
    Let \((a_n)\) be a sequence. We write
    \begin{equation*} \lim_{n \to \infty} a_n = \infty \end{equation*}
    if for every \(c \gt 0\) there exists \(N \in \N\) such that \(n \geq N\) implies
    \begin{equation*} a_n \geq c\text{.} \end{equation*}
    Prove that \(\lim_{n \to \infty} (n^2 - 1) = \infty\text{.}\)
