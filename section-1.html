    Subsection 1.1 Natural numbers and integers

    Axiom 1.1.

    There is a set \(\N\text{,}\) whose elements are called natural numbers, and a function \(s \colon \N \to \N\text{,}\) satisfying:
    1. There is an element \(1 \in \N\text{.}\)
      🔗
    2. There is no \(n \in \N\) such that \(s(n) = 1\text{.}\)
      🔗
    3. The function \(s\) is injective (this means that \(s(n) = s(m)\) only if \(n = m\)).
      🔗
    4. Principle of induction.
      Suppose \(S \subseteq \N\) is a subset such that \(1 \in S\) and if \(n \in S\) then \(n+1 \in S\text{.}\) Then \(S = \N\text{.}\)
      🔗
      🔗
    We call \(s\) the successor function. Informally think of \(s(n) = n+1\text{.}\)
    🔗
    🔗

    Proposition 1.2.

    If \(n \in \N\) and \(n \ne 1\text{,}\) there is a unique \(m \in \N\) such that \(s(m) = n\text{.}\)
    🔗
    🔗

    Proof.

    Let \(A = \{ n \in \N : n = 1 or n = s(m)\text{ for some }m \in N \} \subseteq \N \text{.}\) Then \(1 \in A\) and if \(n \in A\text{,}\) then \(s(n) \in A\text{,}\) so \(A = \N\) by Item 4. Hence, if \(n \in \N\) and \(n \ne 1\text{,}\) then
    \begin{equation*} n = s(m) \end{equation*}
    for some \(m \in \N\text{,}\) which is unique since \(s\) is injective. Indeed, if \(n = s(m')\text{,}\) then \(s(m) = s(m')\text{,}\) therefore
    \begin{equation*} m = m'\text{.} \end{equation*}
    🔗
    🔗

    Remark 1.3.

    We set \(2 = s(1)\text{,}\) \(3 = s(2)\text{,}\) and so on, so that \(\N = \{ 1, 2, 3, \ldots \}\text{.}\)
    🔗
    🔗

    Definition 1.4.

    We define a binary operation \(+ \colon \N \times \N \to \N\) by
    1. For all \(n \in \N\text{,}\) \(n + 1 = s(n)\text{.}\)
      🔗
    2. For all \(n, m \in N\text{,}\) \(n + s(m) = s(n + m)\text{.}\)
      🔗
    🔗
    🔗
    Observe that this is a recursive definition:
    \begin{align*} n+1 \amp = s(n)\\ n+2 \amp = n + s(1) = s(n+1) = s(s(n))\\ n+3 \amp = n + s(2) = s(n+2) = s(n + s(1)) = s(s(n + 1)) = s(s(s(n))) \end{align*}
    🔗

    Proposition 1.5.

    For all \(a, b, c \in \N\text{,}\) we have
    \begin{equation*} (a + b) + c = a + (b + c)\text{.} \end{equation*}
    🔗
    🔗

    Proof.

    Let \(A = \{ c \in \N \colon (a + b) + c = a + (b + c)\text{ for all }a, b \in \N\}\text{.}\) We will show that \(A = \N\) using induction Item 4. Indeed, for \(a, b \in \N\text{,}\) we have
    \begin{equation*} (a + b) + 1 = s(a + b) = a + s(b) = a + (b + 1)\text{,} \end{equation*}
    so \(1 \in A\text{.}\) Now suppose \(c \in A\text{.}\) Then for \(a, b \in \N\text{,}\) we have
    \begin{align*} (a + b) + s(c) \amp = s((a + b) + c) = s(a + (b + c))\\ \amp = a + s(b + c) = a + (b + s(c))\text{.} \end{align*}
    This shows \(s(c) \in A\) and the proof by induction is complete.
    🔗
    🔗

    Definition 1.6.

    We define another operation \(\cdot \colon \N \times \N \to \N\) by:
    1. For all \(n \in \N\text{,}\) \(n \cdot 1 = n\text{.}\)
      🔗
      🔗
    2. For all \(n, m \in \N\text{,}\) \(n \cdot s(m) = n \cdot m + n\text{.}\)
      🔗
      🔗
    🔗
    🔗
    Explicitly,
    \begin{align*} n \cdot 2 \amp = n \cdot s(1) = n \cdot 1 + n = n + n\\ n \cdot 3 \amp = n \cdot s(2) = n \cdot 2 + n = (n + n) + n \end{align*}
    and so on.
    🔗

    Proposition 1.7.

    Let \(a, b, c \in \N\text{.}\)
    🔗
    1. \(\displaystyle a + b = b + a\)
      🔗
      🔗
    2. If \(a + c = b + c\text{,}\) then \(a = b\text{.}\)
      🔗
      🔗
    3. \(a \cdot b = b \cdot a\text{.}\)
      🔗
      🔗
    4. \(a \cdot (b \cdot c) = (a \cdot b) \cdot c\text{.}\)
      🔗
      🔗
    5. \(a \cdot (b + c) = a \cdot b + a \cdot c\text{.}\)
      🔗
      🔗
    6. If \(a \cdot c = b \cdot c\text{,}\) then \(a = c\text{.}\)
      🔗
      🔗
    🔗

    Proof.

    Proof of item 5 only. Let
    \begin{equation*} A = \{ c \in \N : a \cdot (b + c) = a \cdot b + a \cdot c \text{ for all } a, b \in \N \}\text{.} \end{equation*}
    For \(a, b \in \N\text{,}\) we have
    \begin{equation*} a \cdot (b + 1) = a \cdot s(b) = a \cdot b + a = a\cdot b + a\cdot 1\text{,} \end{equation*}
    so \(1 \in A\text{.}\)
    🔗
    Now suppose \(c \in A\) and let \(a, b \in \N\text{,}\) so that \(a \cdot (b+c) = a\cdot b + a \cdot c\text{.}\) Then
    \begin{align*} a \cdot (b + s(c)) \amp = a \cdot s(b + c) = a \cdot (b + c) + a\\ \amp = (a\cdot b + a\cdot c) + a = a\cdot b + a \cdot c + a\\ \amp = a\cdot b + (a\cdot c + a\cdot 1 )= a\cdot b + a(c \cdot 1) = a \cdot b + a\cdot s(c)\text{,} \end{align*}
    which shows \(s(c) \in A\) so that \(A = \N\text{.}\)
    🔗
    🔗
    Now we give the standard form of induction. Let \(P(n)\) be a statement about a natural number \(n\) that is either true or false. If \(P(1)\) is true and if, whenever \(P(n)\) is true we also have \(P(n+1)\) is true, then \(P(n)\) is true for all \(n \in \N\text{.}\)
    🔗

    Definition 1.8.

    If \(m, n \in \N\text{,}\) we say \(m\) is less than \(n\text{,}\) and write \(m \lt n\text{,}\) if \(n = m + k\) for some \(k \in \N\text{.}\) We write \(m \leq n\) if we only wish to assert that \(m \lt n\) or \(m = n\text{.}\)
    🔗
    🔗

    Proposition 1.9.

    Let \(a, b, c \in \N\text{.}\)
    1. We have \(a \leq a\text{,}\) but \(a \not\lt a\text{.}\)
      🔗
      🔗
    2. If \(a \lt b\) and \(b \lt c\text{,}\) then \(a \lt c\text{,}\) and similarly for \(\leq\text{.}\)
      🔗
      🔗
    3. If \(a \leq b\) and \(b \leq a\text{,}\) then \(a = b\text{.}\)
      🔗
      🔗
    4. Exactly one of \(a \lt b\) or \(a = b\) or \(b \lt a\) is true.
      🔗
      🔗
    5. We have \(a \lt b\) if and only if \(a + c \lt b + c\text{.}\)
      🔗
      🔗
    6. We have \(a \lt b\) if and only if \(a \cdot c \lt b \cdot c\text{.}\)
      🔗
      🔗
    🔗
    🔗

    Proof.

    Proof of 2. Suppose that \(a \lt b\) and \(b \lt c\text{.}\) Then we have \(b = a + k\) and \(c = b + \ell\) for some \(k, \ell \in \N\text{.}\) Then
    \begin{equation*} c = b + \ell = (a + k) + \ell = a + (k + \ell) \end{equation*}
    with \(k + \ell \in \N\text{,}\) so \(a \lt c\text{.}\)
    🔗
    Proof of 4. First we will show that no more than one of the statements
    \begin{align*} a \amp \lt b\\ a \amp = b\\ b \amp \lt a \end{align*}
    is true. Suppose first that \(a \lt b\text{.}\) Then \(b = a + k\) for some \(k \in \N\text{.}\) Using the fact that \(m \ne m + n\) for all \(m, n \in \N\text{,}\) we find \(b = a + k \ne a\text{.}\) Similarly, if \(b \lt a\) then \(a \ne b\text{.}\)
    🔗
    Now if \(a \lt b\) and \(b \lt a\) then \(b = a + k\) and \(a = b + \ell\) for some \(k, \ell \in \N\text{,}\) so
    \begin{equation*} b = (b + \ell) + k = b + (\ell + k)\text{,} \end{equation*}
    but this implies \(b \ne b\text{,}\) a contradiction. Similarly, if \(a = b\text{,}\) then \(a \not\lt b\) and \(b \not\lt a\text{.}\) This shows at most one of the statements is true.
    🔗
    To show at most one of the statements is true, let’s fix \(a \in \N\) and let \(P(b)\) be the statement “\(a \lt b\) or \(a = b\) or \(b \lt a\)”. If \(a \ne 1\text{,}\) then \(a = s(k) = 1 + k\) for some \(k \in \N\text{,}\) so \(1 \lt a\) and we have established \(P(1)\text{.}\) Now, let’s suppose that \(b \in \N\) and that \(P(b)\) is true, meaning that \(a \lt b\) or \(a = b\) or \(b \lt a\text{.}\)
    🔗
    Suppose first that \(a \lt b\text{.}\) Then \(b = a + k \) for some \(k \in \N\text{,}\) so \(s(b) = s(a+k) = a + s(k)\text{.}\) This shows that \(a \lt s(b)\text{,}\) so \(P(b+1)\) is true in this case.
    🔗
    Suppose next that \(a = b\text{.}\) Then \(s(b) = b + 1 = a + 1\text{,}\) so \(a \lt s(b)\) and \(P(b+1)\) is true as before.
    🔗
    Finally, suppose that \(b \lt a\text{.}\) Then \(a = b + k\) for some \(k\text{.}\) If \(k = 1\text{,}\) then \(a = b + 1 = s(b)\text{,}\) so \(P(b+1)\) is true. If \(k \ne 1\text{,}\) then \(k = s(\ell)\) for some \(\ell \in \N\) and
    \begin{equation*} a = b + s(\ell) = b + \ell + 1 = (b + 1) + \ell = s(b) + \ell\text{.} \end{equation*}
    This shows that \(a \lt s(b)\text{,}\) and therefore \(P(b+1)\) is true in this case as well.
    🔗
    We have shown that \(P(b) \implies P(b+1)\) for all \(b\in \N\text{,}\) which concludes the proof by induction.
    🔗
    🔗

    Theorem 1.10. The well-ordering principle.

    If \(A \subseteq \N\) is nonempty, then \(A\) has a least element. That is, there exists \(a \in A\) such that \(a \leq b\) for all \(b \in A\text{.}\)
    🔗

    Subsection 1.2 The Integers

    Definition 1.11.

    A relation on a set \(A\) is a subset \(R\) of the Cartesian product \(A \times A\text{.}\) If \((x,y) \in R\text{,}\) we write
    \begin{equation*} x \sim_R y \end{equation*}
    or simply \(x \sim y\) if the relation \(R\) is understood. Sometimes we abuse notation and identify the equivalence operator \(\sim\) with the relation \(R\text{.}\)
    🔗
    🔗

    Definition 1.12.

    A relation \(\sim\) is called an equivalence relation if
    1. It is reflexive: \(x \sim x\) for all \(x \in A\text{.}\)
      🔗
      🔗
    2. It is symmetric: If \(x \sim y\) then \(y \sim x\) for all \(x, y \in A\text{.}\)
      🔗
      🔗
    3. It is transitive: If \(x \sim y\) and \(y \sim z\) then \(x \sim z\) for all \(x, y, z \in A\text{.}\)
      🔗
      🔗
    🔗
    🔗

    Example 1.13.

    Let \(A = \{ 1, 2, 3 \}\) and let \(R = \{ (1, 1), (2, 2), (3, 3), (2, 3), (3, 2) \}\text{.}\) Then observe:
    \begin{align*} (1, 1), (2, 2), (3, 3) \in R \amp \implies \text{reflexive}\\ (2, 3), (3, 2) \in R \amp \implies \text{symmetric}\\ (2, 3), (3, 2) \in R \text{ and } (2, 2) \in R \amp\\ (3, 2), (2, 3) \in R \text{ and } (3, 3) \in R \amp\\ (2, 2), (2, 3) \in R \text{ and } (2, 3) \in R \amp\\ (3, 3), (3, 2) \in R \text{ and } (3, 2) \in R \amp \text{transitive}\text{,} \end{align*}
    so \(R\) is an equivalence relation.
    🔗
    🔗

    Definition 1.14.

    Let \(A\) be a set. A collection of subsets \(A_i\) of \(A\) is called a partition of \(A\) if each element of \(A\) belongs to exactly one of the \(A_i\text{.}\)
    🔗
    🔗
    Notice that Definition 1.14 does not require the elements of a partition to be finite in number. In other words, we can partition a set into infinitely many parts.
    🔗

    Lemma 1.15.

    Suppose \(A\) is partitioned into the subsets \(A_i\text{.}\) Then we have, for all \(i\text{,}\) \(j\text{:}\)
    \begin{equation} A_i \cap A_j = \varnothing \text{ or } A_i = A_j\text{.}\tag{1.1} \end{equation}
    This is usually summarized by saying “The parts of a partition are either equal or disjoint.”
    🔗
    The converse is also true, namely, if \(A_i\) is a collection of subsets of \(A\) satisfying (1.1) and \(\bigcup A_i = A\text{,}\) then the \(A_i\) are a partition of \(A\text{.}\)
    🔗
    🔗

    Proof.

    Suppose first that the \(A_i\) partition \(A\text{.}\) Then each element of \(A\) has to belong to exactly one of the \(A_i\text{.}\) Now suppose \(a\) is an element of \(A_i \cap A_j\text{.}\) Then, because \(a\) belongs to exactly one of the \(A_i\text{,}\) we must have \(A_i = A_j\text{.}\) This shows that the \(A_i\) satisfy (1.1).
    🔗
    On the other hand, suppose that \(A_i\) are some subsets of \(A\) satisfying (1.1). Then let \(a \in A\text{;}\) it is enough to show that \(a\) belongs to exactly one of the \(A_i\text{.}\) If we suppose \(a\) belongs to two of the \(A_i\text{,}\) say \(a \in A_i\) and \(a \in A_j\text{,}\) then by (1.1) we have at once that \(A_i = A_j\text{.}\) So \(a\) cannot belong to more than one of the \(A_i\text{.}\) On the other hand, the union of the \(A_i\) is equal to \(A\text{,}\) hence it certainly contains \(a\text{,}\) which means that \(a\) belongs to at least one of the \(A_i\text{.}\)
    🔗
    Thus we have shown that \(a\) belongs to exactly one of the \(A_i\text{,}\) and so the \(A_i\) are a partition of \(A\text{,}\) as required.
    🔗
    🔗

    Proposition 1.16.

    Let \(\sim\) be an equivalence relation on a set \(A\text{.}\) For each \(a \in A\) we define a subset \([a] = \{ x \in A : x \sim a \}\text{,}\) called the equivalence class of \(a\) under \(\sim\text{.}\) The set of all such equivalence classes is a partition of \(A\text{.}\)
    🔗
    🔗

    Proof.

    We proceed according to Lemma 1.15 and show that the equivalence classes satisfy (1.1) and that their union is \(A\text{.}\)
    🔗
    Suppose \(x \in [a]\) and \(x \in [b]\) for some \(a, b \in A\text{.}\) Then by definition we have \(x \sim a\) and \(x \sim b\text{.}\) Thus \(a \sim x\) and hence \(a \sim b\text{.}\) It follows that
    \begin{equation*} [a] \subseteq [b]\text{.} \end{equation*}
    Indeed, if \(y \sim a\text{,}\) then also \(y \sim x\) (because \(a \sim x\) and \(\sim\) is an equivalence relation). Thence \(y \sim b\) because \(x \sim b\) and finally this means that \(y \in [b]\text{.}\) It follows by a symmetric argument that \([b] \subseteq [a]\text{.}\) Hence \([a] = [b]\text{.}\)
    🔗
    This shows that for all \(a, b \in A\text{,}\) either \([a] \cap [b] = \varnothing\) or \([a] = [b]\text{,}\) which is the content of (1.1).
    🔗
    To see that the union of the equivalence classes is \(A\text{,}\) we just notice that each \(a \in A\) is certainly a member of its own equivalence class \([a]\text{,}\) because \(A\) is an equivalence relation. This concludes the proof.
    🔗
    🔗

    Example 1.17.

    For the equivalence relation \(R\) on the set \(A\) of Example 1.13, namely
    \begin{align*} A \amp = \{ 1, 2, 3 \}\\ R \amp = \{ (1, 1), (2, 2), (3, 3), (2, 3), (3, 2) \}\text{,} \end{align*}
    we have the following equivalence classes:
    \begin{equation*} [1] = \{1\}, [2] = \{2, 3\}, [3] = \{2, 3\}\text{.} \end{equation*}
    The partition of \(A\) associated to the equivalence relation is
    \begin{equation*} A = \{1\} \cup \{2, 3\}\text{.} \end{equation*}
    🔗
    🔗
    Equivalence relations are a powerful method of constructing new sets from old. We can look at the set of equivalence classes as a new mathematical object. This is illustrated in diabolically clever fashion by Definition 1.18.
    🔗

    Definition 1.18.

    We define a relation \(\sim\) on the set \(\N \times \N\) by
    \begin{equation*} (a, b) \sim (c, d) \text{ if } a + d = b + c\text{.} \end{equation*}
    🔗
    This looks strange, but it is concocted quite precisely. We are trying to construct the negative integers out of thin air. The idea is to represent a number like \(-3\) as the class of all pairs \((a, b) \in \N \times \N\) where \(a + 3 = b\text{.}\) It’s the idea of being three less than something.
    🔗
    🔗

    Proposition 1.19.

    The relation \(\sim\) is an equivalence relation.
    🔗
    🔗

    Proof.

    Let \(a, b \in \N\text{.}\) Then \(a + b = b + a\text{,}\) so we have \((a, b) \sim (a, b)\text{.}\) Thus the relation is reflexive.
    🔗
    Next let us suppose that \((a, b) \sim (c, d)\) for some \(a, b, c, d \in \N\text{.}\) Then \(a + d = b + c\text{.}\) It follows that \(c + b = d + a\text{,}\) so \((c, d) \sim (a, b)\text{.}\) Thus the relation is symmetric.
    🔗
    Finally, let us suppose that \((a, b) \sim (c, d)\) and \((c, d) \sim (e, f)\) for some \(a, b, c, d, e, f \in \N\text{.}\) Then \(a + d = b + c\) and \(c + f = d + e\text{.}\) Adding, we find
    \begin{align*} a + d + c + f \amp = b + c + d + e\\ a + f + (c + d) \amp = b + e + (c + d)\\ a + f \amp = b + e \end{align*}
    from which it follows at once that \((a, b) \sim (e, f)\) and the relation is transitive.
    🔗
    🔗

    Definition 1.20.

    The set of integers, denoted \(\Z\text{,}\) is the set of equivalence classes of \(\sim\) in the set \(\N \times \N\text{.}\)
    🔗
    That is,
    \begin{equation*} \Z = \{ [(a, b)] : a, b \in \N \}\text{.} \end{equation*}
    We define the elements \(\overline{0} \in \Z\) and \(\overline{1} \in \Z\) by
    \begin{align*} \overline{0} \amp = [(1, 1)] = \{ (a, b) \in \N \times \N : a = b \} \\ \overline{1} \amp = [(2, 1)] = \{ (a, b) \in \N \times \N : a = 1 + b \}\text{.} \end{align*}
    🔗
    🔗
    Having defined the integers, it’s now our duty to give them their familiar operations and show that these extend the operations on \(\N\text{.}\) This is done in a standard way below.
    🔗

    Definition 1.21.

    We define the operation \(+ \colon \Z \to \Z\) by
    \begin{equation} [(a, b)] + [(c, d)] = [(a + c, b + d)].\tag{1.2} \end{equation}
    This definition requires a short proof to be valid. So it is only provisional for now.
    🔗
    🔗
    For Definition 1.21 to be valid, we have to show that the right-hand side of (1.2) doesn’t depend on our choices of \(a, b, c, d\text{.}\) If we let those numbers vary within their equivalence classes, the right-hand-side must not change. When this is established, we say that addition of integers is well-defined.
    🔗

    Proposition 1.22.

    Let’s suppose that \([(a, b)] = [(a', b')]\text{,}\) so that \((a', b')\) is another member of the equivalence class \([(a, b)]\text{.}\) Similarly, let’s take \((c', d')\) so that \([(c', d')] = [(c, d)]\text{.}\) Then it doesn’t matter which we use to compute \([(a, b)] + [(c, d)]\text{,}\) because
    \begin{equation*} [(a + c, b + d)] = [(a' + c', b' + d')]\text{.} \end{equation*}
    🔗
    🔗

    Proof.

    Since \([(a, b)] = [(a', b')]\) and \([(c, d)] = [(c', d')]\text{,}\) we have
    \begin{equation*} a + b' = b + a' \quad \text{ and } \quad c + d' = d + c'\text{.} \end{equation*}
    Adding, we find
    \begin{align*} a + c + b' + d' \amp = b + d + a' + c'\\ (a + c) + (b' + d') \amp = (b + d) + (a' + c')\text{,} \end{align*}
    which means that \([(a + c), (b + d)] = [(a' + c', b' + d')]\text{,}\) as required.
    🔗
    🔗

    \begin{equation*} [(a, b)] - [(c, d)] = [(a, b)] + (-[(c, d)])\text{.} \end{equation*}

    Definition 1.26.

    Define multiplication \(\cdot \colon \Z \times \Z \to \Z\) by
    \begin{equation*} [(a, b)] \cdot [(c, d)] = [(ac + bd, ad + bc)]\text{.} \end{equation*}

    Proposition 1.27.

    Multiplication is well-defined and satisfies the following for all \(x, y, z \in \Z\text{.}\)
    1. \(\displaystyle (xy)z = x(yz)\)
      🔗
      🔗
    2. \(\displaystyle x \cdot \overline{1} = x\)
      🔗
