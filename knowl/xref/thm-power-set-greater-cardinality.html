Theorem 103.

If \(A\) is and set and \(2^A\) is the power set of \(A\text{,}\) that is
\begin{equation*} 2^A = \{ B : B \subseteq A \}\text{,} \end{equation*}
then \(A\) and \(2^A\) are not equinumerous.

Proof.

Suppose there is a bijection \(f \colon A \to 2^A\text{.}\) Consider the set
\begin{equation*} B = \{ x \in A : x \notin f(x) \}\text{.} \end{equation*}
Since \(B \subseteq A\text{,}\) we have \(B \in 2^A\text{.}\) Since \(f\) is surjective, there is \(y \in A\) such that \(f(y) = B\text{.}\) Now, if \(y \in B\text{,}\) then by the definition of \(B\text{,}\) \(y \notin f(y)\text{,}\) that is, \(y \notin B\text{:}\) clearly, a contradiction.
