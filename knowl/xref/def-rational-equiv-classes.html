Definition 32.

Let \(\Z^{\ast} = \{ x \in \Z : x \ne \overline{0} \}\text{.}\) Define a relation on \(\Z \times \Z^{\ast}\) by
\begin{equation*} (x, y) \sim (z, w) \quad \text{ if } \quad xw = yz\text{.} \end{equation*}
Informally, we think of \((x, y)\) as representing the quotient \(x / y\text{.}\)
