    Theorem 71.

    For any real numbers \(x \lt y\text{,}\) there is a rational number \(r\) with \(x \lt r \lt y\text{.}\)
    ðŸ”—
    ðŸ”—

    Proof.

    Choose \(n \in \N\) so that \(n \gt \frac{1}{y-x}\text{.}\) This is possible by the Archimedean property of the real numbers (TheoremÂ 69). Then
    \begin{equation*} \frac{1}{n} \lt y - x. \end{equation*}
    By the corollary above (CorollaryÂ 70), there is an integer \(m\) such that
    \begin{equation*} m - 1 \leq nx \lt m. \end{equation*}
    Dividing this inequality by \(n\) gives
    \begin{equation*} \frac{m-1}{n} \leq x \lt \frac{m}{n}. \end{equation*}
    Since \(\frac{1}{n} \lt y - x\text{,}\) we have
    \begin{equation*} x + \frac{1}{n} \lt y, \end{equation*}
    and hence
    \begin{equation*} \frac{m}{n} \lt x + \frac{1}{n} \lt y. \end{equation*}
    Thus \(r = \frac{m}{n}\) is a rational number such that \(x \lt r \lt y\text{.}\)

    Proposition 79.

    Suppose \(M \in \R\) is an upper bound for a set \(A \subseteq \R\text{.}\) Then \(M = \sup{(A)}\) if and only if for every \(\eps \gt 0\) there is an \(a \in A\) such that \(M - \eps \lt a\text{.}\)
    ðŸ”—
    ðŸ”—

    Proof.

    Suppose \(M = \sup{(A)}\) and let \(\eps \gt 0\text{.}\) Then \(M - \eps \lt \sup{(A)}\text{,}\) so \(M - \eps\) is not an upper bound for \(A\text{.}\) Therefore, there is an \(a \in A\) such that \(M - \eps \lt a\text{.}\)
    ðŸ”—
    Conversely, suppose that for every \(\eps \gt 0\) there is an \(a \in A\) such that \(M - \eps \lt a\text{.}\) We need to show that \(M\) is the least upper bound for \(A\text{.}\) We already know that \(M\) is an upper bound for \(A\text{,}\) so we need only show that if \(M'\) is any upper bound for \(A\text{,}\) then \(M \leq M'\text{.}\) So let \(M'\) be an upper bound for \(A\text{.}\) If \(M \gt M'\text{,}\) then let \(\eps = M - M' \gt 0\text{.}\) By hypothesis, there is an \(a \in A\) such that \(M - \eps \lt a\text{,}\) or equivalently, \(M' \lt a\text{.}\) But this contradicts the fact that \(M'\) is an upper bound for \(A\text{.}\) Thus \(M \leq M'\text{,}\) and hence \(M = \sup{(A)}\text{.}\)

    Proposition 81.

    There is a positive real number whose square is 2.
    ðŸ”—
    ðŸ”—

    Proof.

    Let \(A = \{ x \in \R : x^2 \lt 2 \}\text{.}\) Then \(A\) is nonempty (since \(1 \in A\)) and bounded above (since \(2\) is an upper bound for \(A\)). By the completeness property of the real numbers (TheoremÂ 77), \(A\) has a supremum; call it \(x = \sup{(A)}\text{.}\) We will show that \(x^2 = 2\text{.}\)
    ðŸ”—
    First suppose that \(x^2 \lt 2\text{.}\) Let \(0 \lt \eps \lt 1\) and compute
    \begin{equation*} (x + \eps)^2 = x^2 + 2x\eps + \eps^2 \lt x^2 + 4\eps + \eps = x^2 + 5\eps\text{,} \end{equation*}
    because \(x \lt 2\) (since \(2\) is an upper bound for \(A\)) and \(\eps \lt 1\text{.}\) Since \(x^2 \lt 2\text{,}\) we can choose \(\eps\) so small that \(x^2 + 5\eps \lt 2\text{.}\) This shows that \((x + \eps)^2 \lt 2\text{,}\) and hence \(x + \eps \in A\text{.}\) But this contradicts the fact that \(x = \sup{(A)}\) and \(\eps \gt 0\text{.}\) Thus \(x^2 \lt 2\) is impossible.
    ðŸ”—
    Now suppose \(x^2 \gt 2\text{.}\) Let \(0 \lt \eps \lt 1\) and compute
    \begin{equation*} (x - \eps)^2 = x^2 - 2x\eps + \eps^2 \geq x^2 - 2x\eps \geq x^2 - 4\eps\text{,} \end{equation*}
    because \(x \lt 2\) (since \(2\) is an upper bound for \(A\)), \(\eps \lt 1\text{,}\) and \(\eps^2 \gt 0\text{.}\) Since \(x^2 \gt 2\text{,}\) we can choose \(\eps\) so small that \(x^2 - 4\eps \gt 2\text{.}\) This shows that \((x - \eps)^2 \gt 2\text{,}\) and hence \(x - \eps \not\in A\text{.}\) But this contradicts the fact that \(x = \sup{(A)}\) and \(\eps \gt 0\text{.}\) Thus \(x^2 \gt 2\) is impossible.
    ðŸ”—
    This shows that \(x^2 = 2\text{,}\) as required.

    Theorem 88.

    For \(n \in \N\text{,}\) let \(a_n \lt b_n\) be real numbers and set
    \begin{equation*} I_n = [a_n, b_n] = \{ x \in \R : a_n \leq x \leq b_n \}\text{.} \end{equation*}
    Suppose that \(I_{n+1} \subset I_n\) for each \(n \in \N\text{,}\) so that
    \begin{equation*} I_1 \supset I_2 \supset I_3 \supset \ldots\text{.} \end{equation*}
    Then the set
    \begin{equation*} \bigcap_{n=1}^\infty I_n \ne \varnothing\text{.} \end{equation*}
    ðŸ”—
    ðŸ”—

    Proof.

    Let \(A = \{ a_n : n \in \N \}\text{.}\) We have \(A \ne \varnothing\) and \(A\) is bounded above by \(b_1\text{,}\) since \(a_n \leq b_n \leq b_1\) for all \(n \in \N\text{.}\) By TheoremÂ 77, \(A\) has a supremum; call it \(x = \sup{(A)}\text{.}\) Now fix \(m \in \N\) and consider \(I_m = [a_m, b_m]\text{.}\) Since \(a_m \in A\text{,}\) we have \(a_m \leq x\text{.}\) Also, since \(b_m\) is an upper bound for \(A\text{,}\) we have \(x \leq b_m\text{.}\) Thus \(x \in I_m\text{.}\) Since \(m\) was arbitrary, we have \(x \in I_n\) for all \(n \in \N\text{,}\) and hence
    \begin{equation*} x \in \bigcap_{n=1}^\infty I_n\text{.} \end{equation*}
    This shows that \(\bigcap_{n=1}^\infty I_n \ne \varnothing\text{,}\) as required.

    Subsection 2.2 Cardinality

    Definition 90.

    Two sets \(A\) and \(B\) are said to be equinumerous if there exists a bijection (one-to-one and onto function) \(f \colon A \to B\text{.}\) In this case, we write \(\abs{A} = \abs{B}\) to denote that the sets have the same cardinality.
    ðŸ”—
    ðŸ”—

    Definition 91.

    A set \(A\) is said to be finite if there exists a natural number \(n \in \N\) such that \(A\) is equinumerous to the set \(\{ 1, 2, \ldots, n \}\text{.}\) In this case, we write \(\abs{A} = n\text{.}\)
    ðŸ”—
    ðŸ”—

    Definition 92.

    A set \(A\) is said to be infinite if it is not finite.
    ðŸ”—
    ðŸ”—

    Example 93.

    The set \(\N\) is infinite. Indeed, if it were finite, then there would be a natural number \(n\) such that \(\N\) is equinumerous to \(\{ 1, 2, \ldots, n \}\text{.}\) This would mean that there is a bijection \(f \colon \{ 1, 2, \ldots, n \} \to \N\text{.}\) But if \(f\) is such a function, then setting
    \begin{equation*} M = max{(f(1), f(2), \ldots, f(n))}\text{,} \end{equation*}
    we have \(M \in \N\) but \(M + 1 \in \N\) and \(M + 1 \not\in \{ f(1), f(2), \ldots, f(n) \}\text{.}\) This contradicts the fact that \(f\) is surjective. Thus \(\N\) is infinite.
    ðŸ”—
    ðŸ”—

    Example 94.

    Define a function \(f \colon \N \to \N \setminus \{1\}\) by
    \begin{equation*} f(n) = n + 1\text{.} \end{equation*}
    Then \(f(m) = f(n)\) implies \(m + 1 = n + 1\text{,}\) and hence \(m = n\text{,}\) so \(f\) is injective. Also, if \(y \in \N \setminus \{1\}\text{,}\) then \(y - 1 \in \N\) and \(f(y - 1) = y\text{,}\) so \(f\) is surjective. Thus \(f\) is a bijection, and hence \(\abs{\N} = \abs{\N \setminus \{1\}}\text{.}\) This shows that it is possible for a set to be equinumerous to a proper subset of itself.
    ðŸ”—
    ðŸ”—

    Example 95.

    Let \(A\) be an infinite set and let
    \begin{equation*} a_1, a_2, a_3, \ldots, a_n \in A \end{equation*}
    be distinct elements of \(A\text{.}\) Such elements exist by the definition of infinite set. Let us show that \(A\) is equinumerous to the set \(A \setminus \{ a_1, \ldots, a_n \}\text{.}\)
    ðŸ”—
    First, we can choose \(a_{n+1}, a_{n+2}, \ldots \in A\) such that
    \begin{equation*} a_1, a_2, \ldots, a_n, a_{n+1}, a_{n+2}, \ldots \end{equation*}
    are all distinct. (If we could not do this, then \(A\) would be finite.) Now define \(f \colon A \to A \setminus \{ a_1, \ldots, a_n \}\) by
    \begin{equation*} f(a_i) = a_{i+n} \text{ for } i = 1, 2, \ldots \end{equation*}
    and
    \begin{equation*} f(x) = x \text{ for } x \in A \setminus \{ a_1, a_2, \ldots \}\text{.} \end{equation*}
    Then \(f\) is a bijection, and hence \(\abs{A} = \abs{A \setminus \{ a_1, \ldots, a_n \}}\text{.}\)

    Example 101.

    The set \(\Q\) of rational numbers is countable. To see this, fix \(m \in \N\) and define
    \begin{equation*} A_m = \left\{ \frac{n}{m} : n \in \N \right\} \subseteq \Q\text{.} \end{equation*}

    \begin{equation*} f \colon \N \to A_m, \quad f(n) = \frac{n}{m} \end{equation*}

    \begin{equation*} \Q = \{0\} \cup \bigcup_{m=1}^\infty (A_m \cup -A_m). \end{equation*}
    Since \(\{0\}\) is finite and each \(A_m\) and \(-A_m\) is countable, it follows from TheoremÂ 100 that \(\Q\) is countable.
